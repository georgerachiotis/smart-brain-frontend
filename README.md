React frontend for the Smart Brain face recognition application.

## Tech Stack
- React
- JavaScript
- CSS
- Fetch API

## Features
- User registration and sign-in
- Face detection on image URLs
- Displays bounding boxes around detected faces
- Tracks user image submission count

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/georgerachiotis/smart-brain-frontend.git
cd smart-brain-frontend
```

### 2. Install dependencies
```bash
npm install
```
### 3. Create a .env file
```env
REACT_APP_API_URL=http://localhost:3000
```

### 4. Start the app
```bash
npm run dev
```
The app will run on http://localhost:3000.
